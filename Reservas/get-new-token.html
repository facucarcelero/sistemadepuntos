🔑 Obtener Token FCM

Esta página te ayudará a obtener un nuevo token FCM para las notificaciones push

📋 Instrucciones:

  1. 1. Abre esta página en tu celular
  2. 2. Haz clic en "Solicitar Permisos de Notificación"
  3. 3. Acepta los permisos cuando el navegador lo solicite
  4. 4. Haz clic en "Obtener Token FCM"
  5. 5. Copia el token y actualízalo en functions/index.js
